# Detilli Hub – Foto → Emoji

App web 100% offline que transforma qualquer foto em mosaico de emojis ou em miniatura pixelada. Feito por Michel Detilli para o projeto Detilli Hub.

## ✨ O que faz
- **Aba Mosaico:** converte a imagem em grade de emojis (😀 ❤️ ⭐ 🎮 📻 etc.)
- **Aba Miniatura:** gera retrato fiel reduzido (32px a 128px) — ideal para figurinha de WhatsApp
- Upload por arquivo, colar da área de transferência ou URL
- Ajustes de brilho, contraste, saturação e filtro
- Fundo transparente ou cor sólida
- Exporta em PNG ou JPG

## 🚀 Como usar
1. Abra https://detiillimichel-max.github.io
2. Envie uma foto
3. Escolha “Mosaico” ou “Miniatura”
4. Ajuste o tamanho (recomendado: 64px = meio polegar)
5. Clique em “Converter” > “Baixar PNG”

## 📱 Tamanho ideal
- Figurinha WhatsApp: 64–96px
- Ícone de canal: 128px
- O app já limita o máximo a 1 polegar para não ficar gigante

Criado com HTML/CSS/JS puro — sem servidor, funciona offline.
